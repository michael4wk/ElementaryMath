# 小学数学精讲项目导航

## 快速入口

- 实施主文档：`README_IMPLEMENTATION.md`
- 文档总览：`docs/README.md`
- Spec 套件总览：`specs/README.md`

## docs 目录

- 交付文档：`docs/delivery/`
- API移交包：`docs/delivery/api/`
- 质量文档：`docs/quality/`
- 运营文档：`docs/operations/`
- 计划文档：`docs/plans/`

## specs 目录

- 质量优化套件：`specs/quality_optimization_v1/`
- 前端交付衔接套件：`specs/frontend_delivery_handoff_v1/`

## 常用产物

- 阶段验收报告：`artifacts/acceptance/report.md`
- 质量门禁报告：`artifacts/quality/gate_report.json`
- 质量趋势看板：`artifacts/ops/quality_trend.json`
- 运维巡检告警：`artifacts/ops/alerts.md`

## 协作约定

- 新增业务说明文档统一放入 `docs/` 对应子目录。
- 新增或迭代 spec 三件套统一放入 `specs/<suite_name>/`。
- 根目录优先保留入口级文件，避免文档分散。
