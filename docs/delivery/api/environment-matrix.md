# API 环境矩阵（v1.1）

| 环境 | Base URL | 用途 | 鉴权 | CORS 策略 | 数据特性 |
|---|---|---|---|---|---|
| local | http://127.0.0.1:18080 | 本地开发自测 | `X-API-Key` | 白名单：`localhost/127.0.0.1` 的 5173/3000 | 本地最新构建产物 |
| dev | 待部署后填写 | 团队联调 | `X-API-Key` | 仅允许前端联调域名 | 近实时同步 |
| staging | 待部署后填写 | 发布前验收 | `X-API-Key` | 最小授权白名单 | 预发数据快照 |
| prod | 待部署后填写 | 生产访问 | `X-API-Key` | 最小授权白名单 + 审计 | 正式在线数据 |

## 配置说明

- 本地默认启动命令（示例）：
  - `python api/minimal_api.py --project-root . --host 127.0.0.1 --port 18080`
- CORS 配置源：`config/security_config.json -> cors`
- 若需跨设备访问，可将 `host` 改为 `0.0.0.0` 并配合网络与防火墙策略。
- 非本地环境地址由部署团队维护并回填本表。

## 联调前检查

- 服务可达：`GET /health` 返回 `code=0`
- 鉴权生效：无密钥访问非白名单接口应返回 `401`
- 追踪可用：响应头包含 `X-Trace-Id`
- 跨域可用：浏览器预检请求（`OPTIONS`）返回成功
