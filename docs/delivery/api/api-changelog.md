# API 变更记录

## v1.1.0（当前）

- 新增前端移交文档包目录：`docs/delivery/api/`
- 新增机器可读接口契约：`openapi.yaml`
- 新增前端接入手册、环境矩阵、示例集合与 Mock 数据
- 新增错误码字典与接口兼容映射表
- 新增前端联调问题记录模板
- 新增契约治理流程文档与一致性校验脚本
- 增加 `GET /problems/{problem_id}` 详情接口
- 增加浏览器跨域支持（CORS + OPTIONS 预检）
- 统一强调 `trace_id` 与响应头 `X-Trace-Id`

## 兼容性说明

- v1.1.0 对既有列表与检索接口保持兼容。
- 变更原则为“新增优先”，不删除既有字段与路径。
